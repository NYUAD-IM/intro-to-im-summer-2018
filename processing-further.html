2. pshapes

a shape is an ensemble of vector points that can be set and manipulated
vectors can be stretched out endlessly without losing quality
processing pdf library

3. pimages

pimages are processing's way to handle images
any external asset (such as images) must be in the sketch's data folder
first you load the image, then you display it using image()

4. objects / classes

an object (or a class, depending on who you ask) is a way to neatly package both variables and function into something that you can re-use easily
an object has variables (fields) and functions (methods)
you can create an object and call its methods directly
e.g. if you have written the code for one bouncing ball, you can turn it into an object and create a lot of independent bouncing balls!

6. working with text

a char is the most basic datatype -it represents 1 character
a string is a collection (well, a string) of characters, and can form sentences
strings are kinds of arrays, which you can join() or splitTokens()

7. text as arrays

you can load external text with loadStrings()
and then you would use for loops to iterate over each string

8. other types of data

CSV (comma-separated values) - loadTable()
JSON (javascript object notation) - loadJSONObject()
XML (extensible markup language) - loadXML()

9. data translation

data as an index of the real (data is linked to reality)
data as formality (data is explored for its own abstraction)
data as information (data is transformed into another type of data)
data as anti-content (data is purged from meaning)
data as datum (data is put into context with other bits of data)
